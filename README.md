# Bluerra Personal Site

A pixelated personal website featuring animated borders, theme switching, and retro aesthetics.

## Credits

The site structure is heavily inspired by [SpectraPulse](https://github.com/spectrapulse), please give him a star ‚≠ê

## Features

- Dark/Light theme toggle
- Animated border effects
- Press Start 2P pixelated font
- Background music support
- Click sound effects
- Responsive design
- Social media links

## Stuff used

- HTML5
- CSS3 (with animations)
- Vanilla JavaScript
- Font Awesome icons
- Google Fonts (Press Start 2P)
